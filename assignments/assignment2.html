Assignment: 2. 3D Prototyping Building Blocks

By: Tuomas Nummela

Date: 28/02/2025

Introduction

In this assignment, we were tasked with designing and creating a 3D prototype based on our previous conceptual design work for creating a custom building block. The objective was to produce something tangible that can be used for physical prototyping.

Tinkering process

Phase 1: Gathering materials in the DesignLab

Gathering different kinds of possible materials for the node blocks and the connecting sticks and tools for tinkering with them. Some required features recognised in the previous assignment were:

  1. The node blocks must accept connector edges from all kinds of angles and allow for quick connecting and disconnecting of edges
  2. The node blocks would ideally be colored and the edges labeled according to weight
  3. The overall structure has to be structurally strong enought to support building elaborate 3D graphs

Phase 2: Selecting the materials for edges

For the edges connecting the node blocks I sharpened the points of different kinds of wooden sticks I found in the Design Lab. I ended up going with the centermost stick since it was the lightest to use and easiest to stick into different kinds of node block candidates. The other two would have offered more surface for possible edge labeling but I did not prioritize that over good connectivity.

Phase 3: Trying out different materials for the node blocks

Now that I had selected the material used for the edges, it was time to try out different materials for the node blocks. Initially I was most enthusiastic about these foam cutouts that were light and allowed easy connectivity to sharpened wooden sticks. Additionally the foam seemed to reconstruct any holes made after disconnecting an edge.

Phase 4: Choosing a material for the node blocks

However, after building some 3D structures with the foam blocks they turned out too smushy to work with which caused the structures to collapse easily. I then turned to these styrofoam-type pellets that allowed easy puncturing with the sticks while still being solid enough to retain structural integrity. In addition they were pre-colored which was perfect for the idea of using the blocks for teaching graph colorings.

Phase 5: Verifying the structural strength of the materials selected

After selecting materials for both the node blocks as well as the edges connecting them, it was time to play around with them a bit more to verify that the structures built using them do not fall apart. The only issue I found was that the styrofoam node blocks do get damaged irreversibly when the edges puncture them. This means that the node blocks used have to be replaced from time to time with new ones. Luckily they do not cost that much and are even be available for anyone to buy themselves too.

Phase 6: Labeling the edges

I thought about different ways of introducing weighted edges into the building kit. The edges themselves are too thin to write anything on, but one solution could be to add stickable or threadable labels, depicted by the tape in the prototype. Labels on edges combined with colored nodes could be used for creating builds such as the one below. The user could then be asked e.g. what is the shortest path from the red node to the yellow node which the user could answer by inspecting the 3D structure hands-on.

Phase 7: Summarizing the kit elements

Altogether the graph building block kit would thus consist of differently colored sets of styrofoam nodes, wooden sticks as the connecting edges and attachable edge weight labels depicted by numbers.

Interoperability

In order to interface the graph building kit with other materials and construction kits, I see two main approaches:

  1. An interface piece that is an edge, i.e. one end is a stick that can be connected to a node and the other is any kind of other building kit piece
  2. An interface piece that is partly a styrofoam node and partly any other type of building kit piece

Out of these two options the edge-interface piece seems more feasible and structurally stable to conscruct since it seems much more difficult to attach anything into styrofoam compared to a wooden stick.
